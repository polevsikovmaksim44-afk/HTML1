1 Заголовки

Заголовки являются одним из важнеших инструментов для структурирования текта

1.1 Рекомендации по использованию

Вот некоторые Рекомендациич по использованию заголовков в html

1.1.1 Использование Н1

Не рекомендуется, чтобы на одной странице содержалось несколько заголовков верхнего уровня

1.1.2 Использование Н2-Н6

При использовании подзаголовков не рекомендуется пропускать уровни заголовков, т.е. после заголовка Н1 должен идти подзаголовок Н2 и только потом Н3 и т.д.
